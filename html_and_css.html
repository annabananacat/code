HTML & CSS

Learn more about HTML & CSS below.

HTML is the foundation for all web pages. It is suggested that you learn this languge first. If HTML did not exist then you would be unable to add images or have your website look fluid. By learning HTML it will allow for you to create templates and to understand how websites work.

CSS is used to make websites look beautiful. It allows for you to create animations if needed or adding gradients and more! It is actually really fun to work with and you can use it to create dynamic responsive websites.
